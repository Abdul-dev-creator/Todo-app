## Todo App

A simple todo list app built with HTML, CSS, and JavaScript.

## Features

- Add a new task with the `Add Task` button
- Add a task by pressing `Enter`
- Empty task validation
- Styled task cards
- Separate Complete and Delete buttons
- Delete task

## Project Files

- `todo.html` - app structure
- `todo.css` - app styling
- `todo.js` - task creation logic

## How to Run

Open `todo.html` in your browser.

## Notes

The app currently creates tasks and displays Complete/Delete buttons. The button styles are ready, but the Complete and Delete button actions still need to be added in JavaScript.
